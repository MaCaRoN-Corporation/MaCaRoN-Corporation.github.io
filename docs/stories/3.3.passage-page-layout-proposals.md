# Propositions de Layouts pour la Page Passage

## Éléments à afficher

Basé sur les stories 3.3-3.8, la page doit afficher :
1. **Timer** (MM:SS) - temps écoulé depuis le début
2. **Compte à rebours** - avant chaque technique (cercle/barre animé)
3. **Technique actuelle** - attaque et technique en grand
4. **Historique** - liste des techniques déjà énoncées (attaque, technique, position)
5. **Indicateur de progression** - "X / Y" techniques + barre de progression
6. **Contrôles** (potentiel) - play/pause/resume (à confirmer)

- Zone centrale prédominante pour la technique actuelle
- Historique en bas, toujours visible mais non intrusif
- Pas de scroll de page, historique avec scroll interne si nécessaire
- Utilise `grid` ou `flex` pour organisation

## Recommandation : Proposition 1 "Dojo Traditionnel"

**Pourquoi cette proposition :**
1. ✅ **Zone centrale dominante** - La technique actuelle est l'élément le plus important
2. ✅ **Historique non intrusif** - En bas, visible mais ne distrait pas
3. ✅ **Organisation logique** - Informations de contrôle en haut, contenu principal au centre
4. ✅ **Adaptable mobile** - Se transforme facilement en layout vertical
5. ✅ **Respecte les contraintes** - Tout visible sans scroll de page
6. ✅ **Esthétique sobre** - Aligné avec l'esprit de l'Aïkido

**Avantages:**
- Hiérarchie visuelle claire
- Focus sur la technique actuelle
- Historique accessible mais discret
- Layout flexible et responsive

## Structure HTML Proposée (Proposition 1)

```html
<div class="passage-page">
  <!-- Zone supérieure: Informations de contrôle -->
  <div class="passage-header">
    <div class="timer-section">[Timer]</div>
    <div class="progress-section">[Progression]</div>
    <div class="controls-section">[Contrôles]</div>
  </div>

  <!-- Zone centrale: Contenu principal -->
  <div class="passage-main">
    <div class="countdown-section">[Compte à rebours]</div>
    <div class="current-technique-section">[Technique actuelle]</div>
  </div>

  <!-- Zone inférieure: Historique -->
  <div class="passage-history">
    [Historique scrollable]
  </div>
</div>
```

**CSS (PC):**
```scss
.passage-page {
  display: grid;
  grid-template-rows: auto 1fr auto;
  height: 100vh;
  // Pas de scroll de page
  overflow: hidden;
}

.passage-header {
  display: flex;
  justify-content: space-between;
  padding: 1rem;
}

.passage-main {
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  flex: 1;
  overflow: hidden;
}

.passage-history {
  max-height: 30vh;
  overflow-y: auto; // Scroll interne uniquement
}
```

**CSS (Mobile):**
```scss
@media (max-width: 767px) {
  .passage-page {
    display: flex;
    flex-direction: column;
    height: 100vh;
    overflow: hidden;
  }
  
  .passage-history {
    flex: 0 1 auto;
    max-height: 40vh;
  }
}
```
